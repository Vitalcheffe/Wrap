# Code of Conduct

## Our standard

WRAP NEBULA is a project for everyone — developers, students, beginners, security researchers, people who just want a personal AI that respects their privacy.

We expect everyone participating in this project (issues, PRs, discussions) to:

- Be respectful and constructive
- Welcome people of all backgrounds and experience levels
- Give useful feedback, not personal criticism
- Accept feedback gracefully

## Not acceptable

- Harassment, insults, or personal attacks
- Discriminatory language or behavior
- Sharing private information without consent
- Any behavior that would make someone feel unwelcome

## Enforcement

Issues can be reported to the maintainers via a private message or by emailing the address in the security policy. Violations may result in removal from the project.

---

This is intentionally short. Be kind, build good things. 🌌
