Built different.

Every component is designed for security-first operation. Independent layers. Compromising one doesn't compromise the others.

🔒

V8 Sandbox

Code runs in an isolated environment. Temp directory, stripped environment variables, 128MB memory limit, 30-second timeout. Your code can't touch the host system.

✍️

Ed25519 Audit

Every action is cryptographically signed and appended to an immutable log. Hash-chained. Verify the entire history with one function call.

🛡️

Input Sanitizer

Prompt injection (10 patterns), SQL injection (6), XSS (5). PII like SSN, credit cards, and emails are automatically redacted.

🏠

100% Local

Ollama runs on your machine. No data leaves your computer. No API keys needed. No cloud dependency. Free forever.

🔌

Multi-Provider

Works with Ollama (local), Claude (Anthropic), and GPT-4 (OpenAI). Auto-detected from config. Switch anytime.

📦

Plugin System

Create, install, and share skills. Each skill is hash-verified before loading. nebula skill create my-skill

🧠

SQLite Memory

Conversations stored locally in SQLite. Never transmitted. Never synced. Your data stays on your machine. Period.

⚡

Zero Config

SOUL.md — your agent personality in simple markdown. No YAML schemas. No JSON configs. Just markdown.

🤖

Sub-Agents

Ed25519-signed message bus. Task delegation based on skills. Each agent has its own identity and context.

Full control. No terminal required.

Visual interface to manage agents, monitor activity, create sub-agents.
