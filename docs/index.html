What it does.

Security-first. Every component independent. Compromising one layer does not compromise the others.

V8

Sandbox

Code runs isolated. Temp directory, stripped environment, 128MB memory, 30-second timeout.

SIG

Audit Trail

Ed25519-signed. Verify entire history with one call. Immutable, hash-chained log.

FLT

Sanitizer

Prompt injection, SQL, XSS blocked. PII auto-redacted before reaching the LLM.

LOC

100% Private

Ollama on your machine. No data leaves. No API keys needed. Free forever.

MLT

Multi-Provider

Ollama, Claude, GPT-4. Auto-detected from config. Switch anytime.

PKG

Plugins

Create, install, share skills. Hash-verified. nebula skill create

SQL

Memory

SQLite. Never transmitted. Never synced. Your data stays on your machine.

SOL

SOUL.md

Agent personality in markdown. No YAML. No JSON. Just markdown.

NET

Multi-Agent

Signed message bus. Task delegation. Each agent has its own identity.
