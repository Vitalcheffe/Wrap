What it does.

Security-first. Every component independent. Compromising one layer does not compromise the others.

V8

Sandbox

Code runs in an isolated environment. Temp directory, stripped environment, 128MB memory limit, 30-second timeout.

Ed25519

Audit Trail

Every action is cryptographically signed. Verify the entire history with one function call. Immutable log.

Shield

Input Sanitizer

Prompt injection, SQL injection, XSS detected and blocked. PII auto-redacted before reaching the LLM.

Local

100% Private

Ollama runs on your machine. No data leaves your computer. No API keys needed. Free forever.

Multi

Provider

Works with Ollama, Claude, and GPT-4. Auto-detected from config. Switch anytime.

Plugin

System

Create, install, and share skills. Hash-verified before loading. nebula skill create

SQL

Memory

Conversations stored in SQLite. Never transmitted. Never synced. Your data stays on your machine.

MD

SOUL.md

Agent personality in simple markdown. No YAML. No JSON configs. Just markdown.

Net

Multi-Agent

Ed25519-signed message bus. Task delegation. Each agent has its own identity.

Security.

Independent layers. Each one protects against a different class of failure.

01
Input SanitizerDetects prompt injection (10 patterns), SQL injection (6), XSS (5). PII redacted automatically.
02
Rust GovernorSeparate process with own memory. Binary-level policy enforcement. Audit signing.
03
V8 SandboxTemp directory. Minimal PATH. No network. 128MB memory. 30-second timeout.
04
Ed25519 Audit TrailAppend-only, hash-chained, cryptographically signed. Verify all with one call.
05
SQLite MemoryLocal only. Never transmitted. Never synced. Your data stays on your machine.
