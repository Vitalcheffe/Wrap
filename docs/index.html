Built different.

Every component is designed for security-first operation.

🔒

V8 Sandbox

Code execution runs in an isolated environment. Temp directory, stripped environment variables, blocked dangerous patterns, output limits. Your code can't touch the host system.

✍️

Ed25519 Audit Trail

Every action is cryptographically signed. Verify the entire history with verifyAll(). Immutable, append-only, tamper-evident.

🛡️

Input Sanitizer

Prompt injection, SQL injection, XSS — detected and blocked before reaching the LLM. PII (SSN, credit cards, emails) auto-redacted.

🏠

100% Local

Ollama runs on your machine. No data leaves your computer. No API keys needed. No cloud dependency. Free forever.

🔌

Multi-Provider

Works with Ollama (local), Claude (Anthropic), and GPT-4 (OpenAI). Auto-detected from your config. Switch anytime.

📦

Plugin System

Create, install, and share skills. Each skill is hash-verified before loading. nebula skill create my-skill

Security architecture.

Every layer is independent. Compromising one doesn't compromise the others.

01
Input Sanitizer
Prompt injection (10 patterns), SQL injection (6), XSS (5). PII redacted automatically.
02
Rust Governor
Separate process. Validates permissions, filters injections, signs audit entries at binary level.
03
V8 Sandbox
Temp HOME, minimal PATH, no network, 128MB memory, 30s timeout. Dangerous patterns blocked.
04
Ed25519 Audit Trail
Append-only, hash-chained, cryptographically signed. Verify all with one function call.
05
SQLite Memory
Local only. Never transmitted. Never synced. Your data stays on your machine.
