What it does.

Security-first. Every component independent. Compromising one layer does not compromise the others.

V8

Sandbox

Code runs in an isolated environment. Temp directory, stripped environment, 128MB memory, 30-second timeout.

25519

Audit Trail

Every action is cryptographically signed. Verify the entire history with one function call. Immutable.

INJ

Sanitizer

Prompt injection, SQL injection, XSS detected and blocked. PII auto-redacted before reaching the LLM.

LOC

100% Private

Ollama runs on your machine. No data leaves your computer. No API keys needed. Free forever.

MUL

Multi-Provider

Works with Ollama, Claude, and GPT-4. Auto-detected from config. Switch anytime.

PKG

Plugins

Create, install, and share skills. Hash-verified before loading. nebula skill create

SQL

Memory

Conversations stored in SQLite. Never transmitted. Never synced. Your data stays on your machine.

SOL

SOUL.md

Agent personality in simple markdown. No YAML schemas. No JSON configs. Just markdown.

NET

Multi-Agent

Ed25519-signed message bus. Task delegation based on skills. Each agent has its own identity.

Security.

Independent layers. Each one protects against a different class of failure.

01
Input SanitizerPrompt injection (10 patterns), SQL injection (6), XSS (5). PII redacted.
02
Rust GovernorSeparate process with own memory. Binary-level policy enforcement.
03
V8 SandboxTemp directory. Minimal PATH. No network. 128MB. 30s timeout.
04
Ed25519 TrailAppend-only, hash-chained, signed. Verify all with one function call.
05
SQLite MemoryLocal only. Never transmitted. Never synced. Period.
